# 종합소득세 신고 절차 완벽 정리

프리랜서와 소상공인을 위한 종합소득세 신고 절차를 단계별로 상세히 설명합니다. 홈택스 사용법부터 서류 준비, 실수 방지 팁까지 모든 것을 알려드립니다.

## 신고 기한 및 일정

### 신고 기한

- **기간**: 매년 5월 1일 ~ 5월 31일
- **기한 연장**: 불가 (법정 기한)
- **가산세**: 기한 경과 시 가산세 부과

### 신고 일정표

| 일정 | 내용 |
|------|------|
| **3월** | 소득금액 증명원류 발급 |
| **4월** | 경비 증빙 정리, 공제 항목 확인 |
| **5월 1일~31일** | 홈택스 신고 |
| **6월** | 납부 또는 환급 |

## 신고 전 준비사항

### 1. 서류 준비 체크리스트

#### 필수 서류

- [ ] 소득금액 증명원류 (의뢰인별)
- [ ] 원천징수 영수증
- [ ] 사업자등록증 (사업자 등록한 경우)

#### 경비 증빙 (장부기장 사용 시)

- [ ] 사무실 임대료 영수증
- [ ] 통신비 영수증
- [ ] 교통비 영수증
- [ ] 소모품 구매 영수증
- [ ] 교육비 영수증
- [ ] 기타 업무 관련 경비 영수증

#### 공제 증빙

- [ ] 의료비 영수증
- [ ] 교육비 영수증
- [ ] 기부금 영수증
- [ ] 보험료 납입 증명서
- [ ] 퇴직연금 납입 증명서

### 2. 정보 확인

- [ ] 연 매출 총액 확인
- [ ] 원천징수세액 확인
- [ ] 부양가족 수 확인
- [ ] 공제 항목 확인

## 홈택스 신고 절차

### Step 1: 홈택스 접속

1. [홈택스](https://www.hometax.go.kr) 접속
2. 공동인증서 로그인
3. **종합소득세 신고** 메뉴 선택

### Step 2: 신고서 작성

#### 기본 정보 입력

- 납세자 정보 확인
- 부양가족 정보 입력
- 거주지 정보 확인

#### 소득 정보 입력

1. **사업소득** 선택
2. **단순경비율** 또는 **장부기장** 선택
3. 매출액 입력
4. 경비 입력 (장부기장 사용 시)

#### 공제 정보 입력

- 기본공제
- 부양가족공제
- 의료비공제
- 교육비공제
- 기부금공제
- 보험료공제

#### 세액 정보 확인

- 산출세액 확인
- 원천징수세액 확인
- 결정세액 확인
- 환급금/추가납부 확인

### Step 3: 신고서 검토

- [ ] 입력 정보 정확성 확인
- [ ] 계산 결과 확인
- [ ] 공제 항목 누락 확인
- [ ] 오류 메시지 확인

### Step 4: 신고서 제출

1. **전자신고** 선택
2. 신고서 제출
3. 제출 완료 확인서 출력

## 실수 방지 팁

### 자주 하는 실수

1. **매출액 오입력**
   - 해결: 소득금액 증명원류와 대조 확인

2. **공제 항목 누락**
   - 해결: 공제 체크리스트 확인

3. **원천징수세액 오입력**
   - 해결: 원천징수 영수증 확인

4. **부양가족 수 오입력**
   - 해결: 가족관계증명서 확인

### 검증 방법

1. **계산기로 미리 계산**
   - [종합소득세 계산기](/calculator)로 예상 세액 확인

2. **이전 연도와 비교**
   - 전년도 신고서와 비교하여 이상 여부 확인

3. **세무사 상담**
   - 복잡한 경우 세무사 상담 권장

## 납부 및 환급

### 추가납부

- **납부 기한**: 신고 기한과 동일 (5월 31일)
- **납부 방법**: 
  - 홈택스 온라인 납부
  - 가상계좌 입금
  - 세무서 방문 납부

### 환급금

- **입금 시기**: 신고 후 약 2-3주
- **입금 계좌**: 신고서에 기재한 계좌
- **확인 방법**: 홈택스에서 환급금 조회

## 신고 후 확인사항

### 확인 항목

- [ ] 신고서 제출 완료 확인
- [ ] 납부 완료 확인 (추가납부인 경우)
- [ ] 환급금 입금 확인 (환급인 경우)
- [ ] 가산세 부과 여부 확인

### 문제 발생 시

- **신고서 수정**: 홈택스에서 수정 신고 가능
- **이의신청**: 세무서에 이의신청 가능
- **세무사 상담**: 복잡한 경우 전문가 상담

## 주의사항

### ⚠️ 기한 준수

신고 기한을 지키지 않으면 가산세가 부과됩니다. 반드시 5월 31일까지 신고하세요.

### ⚠️ 정확한 정보 입력

입력 정보가 부정확하면 세무 조사 대상이 될 수 있습니다. 신중하게 입력하세요.

### ⚠️ 증빙 보관

신고 후 5년간 모든 증빙 자료를 보관하세요. 세무 조사 시 요구될 수 있습니다.

## 결론

종합소득세 신고는 복잡해 보이지만, 체계적으로 준비하면 누구나 할 수 있습니다. 이 가이드를 참고하여 자신 있게 신고하시기 바랍니다.

**신고 전 체크리스트:**
1. 서류 준비 완료
2. [세금 계산기](/calculator)로 예상 세액 확인
3. 홈택스에서 신고서 작성
4. 신고서 검토 및 제출
5. 납부 또는 환급 확인
